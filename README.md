# 勿繁 · 短诗集

> *写字的人,留下一些短记。*
> *不归类、不解释、不追认。*

**94 首短诗,写于 2015–2019。**
每一首配一张图。

🌐 **在线阅读:[young920.github.io/wufan-poetry](https://young920.github.io/wufan-poetry/)**

---

## 这是什么

一个单页静态网站,陈列作者本人在 2015 到 2019 这五年里写下的 94 首短诗。

每一首诗配一张 **马格利特(René Magritte)风格** 的画面 —— 写实的人、物、空间,配以荒诞的错位与悬空,作为诗的"另一种翻译"。诗与图互为注脚,谁也不是谁的插图。

不分类。年份就是唯一的顺序。
不解释。诗在那里,自取。
不追认。写于那时,就是那时。

## 设计

- **配色**:宣纸暖白 `#F5EDE4` + 深褐墨色 `#2C1810`,通篇不加高饱和,只用纸、墨、印章红三色隐喻。
- **字体**:`Noto Serif SC`(正文)+ `Cormorant Garamond`(英文与数字)。
- **质感**:页面覆一层极淡的纹理底纹,模拟宣纸手感。导航栏半透明毛玻璃,圆角长条。
- **版式**:`精选` 顶置轮播 + `全部诗作` 瀑布网格 + `关于` 极简 meta。诗的正文居中、单栏、最大宽 600px。
- **单文件**:`index.html` 内联 CSS 与数据(JS 数组),无构建步骤、无依赖。

## 浏览

直接打开 `index.html`,或者访问 [GitHub Pages 站点](https://young920.github.io/wufan-poetry/)。

```
/
├── index.html        # 单文件站点(HTML + CSS + JS 数据)
├── images/           # 95 张诗配图 + 1 张纹理底图 (14.7 MB)
└── README.md
```

诗与配图的数据结构(在 `index.html` 内):

```js
{
  title: '我想',                          // 诗题
  date: '2015.5.30',                      // 写作日期
  content: '将悲哀写进命里，用血去颂扬。...',  // 正文,\n 换行
  image: 'images/poem_001.jpg'            // 配图
}
```

## 技术
